What is GIS & Remote Sensing?

Understand the foundations and advanced concepts of geospatial science.

🗺️

Geographic Information Systems (GIS)

GIS is a framework for gathering, managing, and analyzing data rooted in geography. It integrates many types of data and uses geography as the common thread to visualize, question, analyze, and interpret data to understand relationships, patterns, and trends.

  • Spatial data management & querying
  • Vector & raster data models
  • Coordinate systems & projections
  • Geoprocessing & spatial analysis
  • Map design & cartography

Core Components of GIS

Hardware: Computers, GPS devices, scanners, digitizers, and plotters that enable data collection and output.

Software: ArcGIS, QGIS, GRASS GIS, MapInfo — tools to input, store, query, analyze, and display data.

Data: Spatial data (maps, satellite images) and attribute data (tables, databases) describing the geographic features.

Methods: Overlay analysis, buffer zones, network analysis, terrain modeling, interpolation.

People: GIS specialists, analysts, cartographers, and end users who use the system for decisions.

Applications

Urban planning, transportation routing, disaster management, environmental monitoring, agriculture, public health, defense, and natural resource management.

🛰️

Remote Sensing

Remote sensing is the science of acquiring information about the Earth's surface without physical contact, typically through satellite or aircraft-mounted sensors that detect electromagnetic radiation.

  • Active vs. Passive sensors
  • Electromagnetic spectrum & bands
  • Image preprocessing & corrections
  • Classification & change detection
  • Vegetation, water & land indices

Electromagnetic Spectrum in Remote Sensing

Visible (0.4–0.7 µm): Natural color imagery for land cover mapping and urban analysis.

Near-Infrared (0.7–1.3 µm): Vegetation health, biomass estimation (NDVI applications).

SWIR (1.3–3 µm): Soil moisture, mineral detection, fire mapping.

Thermal IR (3–14 µm): Land surface temperature, urban heat islands, volcanic activity.

Microwave (1mm–1m): Penetrates clouds, used in SAR for surface deformation and flood mapping.

Key Indices

NDVI (vegetation), NDWI (water), NDBI (built-up), EVI (enhanced vegetation), SAVI (soil adjusted), NBR (burn ratio), LST (land surface temperature).

📷

Photogrammetry & Drone Mapping

Photogrammetry uses photographs to measure and interpret geographic features. Modern UAV/drone-based photogrammetry revolutionizes high-resolution 3D terrain modeling.

  • Structure from Motion (SfM)
  • Point cloud processing
  • Digital Elevation Models (DEM/DSM)
  • Orthophoto generation
  • 3D mesh & texture mapping

Drone Surveying Workflow

1. Mission Planning: Define flight area, altitude, overlap (front 80%, side 70%), GSD requirements.

2. Ground Control Points (GCPs): Establish surveyed reference points for georeferencing accuracy.

3. Data Acquisition: Execute automated flight with camera triggering at planned intervals.

4. Processing (SfM): Import images into Agisoft Metashape, Pix4D, or ODM for alignment, dense cloud, mesh, and orthophoto generation.

5. Validation: Check accuracy against GCPs, export deliverables (GeoTIFF, LAS, DXF).

📊

Geospatial Analysis

Spatial analysis turns raw data into actionable insights using mathematical and statistical methods on geographic features and their relationships.

  • Overlay & proximity analysis
  • Network analysis (routing, service areas)
  • Terrain analysis (slope, aspect, viewshed)
  • Spatial interpolation (IDW, Kriging)
  • Time-series & change analysis

Common Analysis Techniques

Buffer Analysis: Create zones of specified distances around features to assess spatial relationships.

Overlay Analysis: Combine multiple layers (union, intersect, erase) to find spatial coincidences or conflicts.

Raster Algebra: Perform mathematical operations on raster grids — calculating NDVI, slope, or weighted suitability models.

Hot Spot Analysis: Detect statistically significant spatial clusters using Getis-Ord Gi* statistics.

Least Cost Path: Identify optimal routes across terrain considering friction surfaces.

🌍

Major Satellite Platforms

Dozens of Earth observation satellites provide multispectral, hyperspectral, SAR, and thermal imagery for diverse geospatial applications.

  • Landsat 8/9 (30m multispectral)
  • Sentinel-1/2/3 (ESA, free access)
  • MODIS (daily global coverage)
  • WorldView, Pleiades (sub-meter)
  • SRTM, ASTER (elevation data)

Platform Comparison

Landsat 8/9: 30m multispectral, 15m pan, 16-day revisit. Best for land cover, agriculture, water body mapping.

Sentinel-2: 10/20/60m multispectral, 5-day revisit, 13 bands. Ideal for vegetation, urban, coastal monitoring.

Sentinel-1 SAR: C-band SAR, cloud penetrating, all-weather. Used for flood mapping, subsidence, crop monitoring.

MODIS: 250m–1km, daily global, 36 bands. Perfect for fire detection, ocean color, climate variables.

Commercial (DigitalGlobe/Maxar): 30cm resolution for precision mapping, infrastructure assessment.

AI & Machine Learning in GIS

Machine learning and deep learning are transforming geospatial analysis — automating feature extraction, image classification, and predictive modeling at scale.

  • Image segmentation & object detection
  • Land cover classification (RF, SVM, CNN)
  • Change detection with deep learning
  • Predictive spatial modeling
  • Google Earth Engine ML integration

ML Approaches for Remote Sensing

Random Forest: Ensemble classifier widely used for land use/land cover (LULC) mapping. Handles high-dimensional data, resistant to overfitting.

Support Vector Machine (SVM): Effective for hyperspectral image classification and small training datasets.

Convolutional Neural Networks (CNN): Deep learning for object detection (buildings, roads, vehicles) in high-resolution imagery.

U-Net: Semantic segmentation architecture widely used for satellite image segmentation and flood mapping.

Transformers: Vision Transformers (ViT) for large-scale geospatial foundation models (Prithvi, SkySense).

Python Calculate NDVI 
import rasterio
import numpy as np

def calculate_ndvi(red_path, nir_path, output_path):
    """Calculate NDVI from Red and NIR bands."""
    with rasterio.open(red_path) as red_ds:
        red = red_ds.read(1).astype(float)
        profile = red_ds.profile

    with rasterio.open(nir_path) as nir_ds:
        nir = nir_ds.read(1).astype(float)

    # Avoid division by zero
    np.seterr(divide='ignore', invalid='ignore')
    ndvi = np.where(
        (nir + red) == 0,
        0,
        (nir - red) / (nir + red)
    )

    profile.update(dtype=rasterio.float32, count=1, nodata=-9999)
    with rasterio.open(output_path, 'w', **profile) as dst:
        dst.write(ndvi.astype(rasterio.float32), 1)

    print(f"NDVI saved to {output_path}")
    print(f"Min: {ndvi.min():.4f}, Max: {ndvi.max():.4f}")
    return ndvi

# Usage
ndvi = calculate_ndvi('red.tif', 'nir.tif', 'ndvi.tif')

Python Batch Reproject Rasters 
import os
import rasterio
from rasterio.warp import calculate_default_transform, reproject, Resampling

def batch_reproject(input_folder, output_folder, target_crs='EPSG:4326'):
    """Batch reproject all TIF files in a folder."""
    os.makedirs(output_folder, exist_ok=True)
    
    tif_files = [f for f in os.listdir(input_folder) if f.endswith('.tif')]
    print(f"Found {len(tif_files)} files to reproject")
    
    for fname in tif_files:
        src_path = os.path.join(input_folder, fname)
        dst_path = os.path.join(output_folder, f"reproj_{fname}")
        
        with rasterio.open(src_path) as src:
            transform, width, height = calculate_default_transform(
                src.crs, target_crs, src.width, src.height, *src.bounds
            )
            kwargs = src.meta.copy()
            kwargs.update({'crs': target_crs,
                           'transform': transform,
                           'width': width,
                           'height': height})
            
            with rasterio.open(dst_path, 'w', **kwargs) as dst:
                for i in range(1, src.count + 1):
                    reproject(
                        source=rasterio.band(src, i),
                        destination=rasterio.band(dst, i),
                        src_transform=src.transform,
                        src_crs=src.crs,
                        dst_transform=transform,
                        dst_crs=target_crs,
                        resampling=Resampling.bilinear
                    )
        print(f"  ✓ {fname} → {target_crs}")

# Usage
batch_reproject('./input_rasters', './output_wgs84', 'EPSG:4326')

Earth Engine SAR Flood Mapping 
// Sentinel-1 SAR Flood Detection
var geometry = ee.Geometry.Rectangle([88.0, 24.0, 92.0, 27.0]); // Bangladesh

var before = ee.ImageCollection('COPERNICUS/S1_GRD')
  .filterBounds(geometry)
  .filterDate('2024-05-01', '2024-06-01')
  .filter(ee.Filter.listContains('transmitterReceiverPolarisation', 'VV'))
  .filter(ee.Filter.eq('instrumentMode', 'IW'))
  .select('VV').mean();

var after = ee.ImageCollection('COPERNICUS/S1_GRD')
  .filterBounds(geometry)
  .filterDate('2024-07-01', '2024-08-01')
  .filter(ee.Filter.listContains('transmitterReceiverPolarisation', 'VV'))
  .filter(ee.Filter.eq('instrumentMode', 'IW'))
  .select('VV').mean();

// Apply speckle filter
var smoothing = 100;
var before_s = before.focal_mean(smoothing, 'circle', 'meters');
var after_s  = after.focal_mean(smoothing, 'circle', 'meters');

// Threshold for flood detection
var diff = after_s.subtract(before_s);
var flood = diff.lt(-3); // >3dB decrease = flood

Map.centerObject(geometry, 7);
Map.addLayer(before_s, {min:-25,max:0}, 'Before (SAR)');
Map.addLayer(after_s,  {min:-25,max:0}, 'After (SAR)');
Map.addLayer(flood.updateMask(flood), {palette:['0000FF']}, 'Flood Extent');

var area = flood.multiply(ee.Image.pixelArea())
               .reduceRegion({reducer:ee.Reducer.sum(), geometry:geometry, scale:10})
               .get('VV');
print('Flooded Area (m²):', area);

Python Dissolve & Clip Shapefile 
import geopandas as gpd

def dissolve_by_attribute(input_shp, output_shp, dissolve_field):
    """Dissolve shapefile by attribute field."""
    gdf = gpd.read_file(input_shp)
    print(f"Input: {len(gdf)} features, CRS: {gdf.crs}")
    
    dissolved = gdf.dissolve(by=dissolve_field, aggfunc='sum')
    dissolved = dissolved.reset_index()
    
    dissolved.to_file(output_shp)
    print(f"Output: {len(dissolved)} features saved to {output_shp}")
    return dissolved

def clip_vector_by_polygon(input_shp, clip_shp, output_shp):
    """Clip vector layer by polygon boundary."""
    gdf = gpd.read_file(input_shp)
    clip = gpd.read_file(clip_shp)
    
    # Ensure same CRS
    if gdf.crs != clip.crs:
        clip = clip.to_crs(gdf.crs)
    
    clipped = gpd.clip(gdf, clip)
    clipped.to_file(output_shp)
    print(f"Clipped: {len(clipped)} features retained")
    return clipped

# Usage
dissolve_by_attribute('districts.shp', 'states.shp', 'STATE_NAME')
clip_vector_by_polygon('roads.shp', 'study_area.shp', 'roads_clipped.shp')

Python Random Forest LULC Classification 
import numpy as np
import rasterio
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report

def classify_lulc(image_path, training_csv, output_path):
    """Random Forest LULC classification from multiband raster."""
    import pandas as pd
    
    # Load multiband image
    with rasterio.open(image_path) as src:
        img_data = src.read()  # (bands, rows, cols)
        profile = src.profile
        n_bands, n_rows, n_cols = img_data.shape
    
    # Reshape: (pixels, bands)
    X_all = img_data.reshape(n_bands, -1).T
    
    # Load training data (columns: B1,B2,...,Bn, Label)
    df = pd.read_csv(training_csv)
    X_train = df.drop('Label', axis=1).values
    y_train = df['Label'].values
    
    # Split for validation
    X_tr, X_val, y_tr, y_val = train_test_split(
        X_train, y_train, test_size=0.3, random_state=42, stratify=y_train
    )
    
    # Train Random Forest
    clf = RandomForestClassifier(n_estimators=100, n_jobs=-1, random_state=42)
    clf.fit(X_tr, y_tr)
    
    # Validate
    y_pred = clf.predict(X_val)
    print(classification_report(y_val, y_pred))
    
    # Classify full image
    classified = clf.predict(X_all).reshape(n_rows, n_cols)
    
    # Save result
    profile.update(dtype=rasterio.uint8, count=1, nodata=0)
    with rasterio.open(output_path, 'w', **profile) as dst:
        dst.write(classified.astype(rasterio.uint8), 1)
    
    print(f"Classification saved: {output_path}")
    return clf, classified

# Usage
clf, result = classify_lulc('sentinel2_6band.tif', 'training.csv', 'lulc_output.tif')
